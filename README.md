# FiveM + React + Tailwind = ❤️

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

After your application ready run the build command:

```bash
npm run build
# or
yarn build
# or
pnpm build
# or
bun build
```

And

```bash
refresh react-boilerplate-tailwind
ensure react-boilerplate-tailwind
```
